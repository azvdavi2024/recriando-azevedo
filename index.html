Utilize a Tag header

Usando o Header

O header é utilizado para representar o cabeçalho de um documento ou seção declarado no HTML. Nele podemos inserir elementos da tag h1 a h6, ate elementos para representar imagens,parágrafos ou mesmo listas de navegação.

Utilize a Tag article

Usando a Tag article

Utilizamos a tag article quando precisamos declarar um conteudo que nao precisa de outro para fazer sentido em um documento HTML.

Utilize a Tag section

Usando a Tag section

O elemento section representa uma ação dentro de um documento e geralmente contém um titulo, o qual e definio por meio de um dos elementos entre h1 e h6.
